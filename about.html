About the Project

CardioAI

An open-source cardiovascular health claim verification system powered by BioBERT and built for research, education, and clinical awareness.

Mission

Why CardioAI?

Cardiovascular disease is the leading cause of death globally. Yet patients and caregivers are bombarded with conflicting, often misleading health claims — from social media posts to wellness blogs.

CardioAI bridges the gap between raw health misinformation and evidence-based cardiology by providing an instant, AI-driven fact-check on cardiovascular claims.

17.9M CV disease deaths per year globally (WHO)
78% Of adults encounter health misinformation online
3x Faster decision-making with AI-assisted triage
True Supported by evidence
False Contradicts research
Uncertain Mixed or emerging evidence
Features

What CardioAI Offers

Voice + Text Input

Submit claims by speaking or typing. The Web Speech API transcribes voice in real time without any third-party service.

Cardio-Domain Only

Laser-focused on cardiovascular medicine. No general-medicine noise — just coronary disease, hypertension, cholesterol, lifestyle, and more.

Confidence Scores

Each prediction comes with a calibrated confidence percentage so users know how certain the model is.

Plain-English Explanations

Not just a label — the model returns a short, lay-friendly explanation of why a claim is true, false, or uncertain.

Fully Responsive UI

Optimized for mobile, tablet, and desktop. The interface adapts seamlessly across all screen sizes.

REST API

Expose predictions via /api/predict — integrate CardioAI into any external app, dashboard, or chatbot.

Roadmap

Future Improvements

PubMed Citation Linking

Automatically link the verdict to the most relevant PubMed papers supporting or refuting the claim.

Multi-language Support

Extend to Spanish, French, Hindi, and Arabic using multilingual BioBERT variants.

EHR Integration

Connect to FHIR-compliant Electronic Health Records for patient-context-aware predictions.

Larger Dataset (50K+)

Scale training data using automated PubMed scraping and expert annotation pipelines.

Explainability (SHAP/LIME)

Highlight which words in the claim most influenced the prediction using token-level attribution.
