Preprocessing Pipeline

Raw text goes through several NLP cleaning steps before tokenization.

  • Lowercase normalization
  • Remove HTML tags & special characters
  • Expand medical abbreviations
  • Sentence tokenization (NLTK)
  • BioBERT WordPiece tokenization
  • Padding / truncation to 512 tokens
  • Attention mask generation

Pre-trained on PubMed

BioBERT is initialized on 18B+ words from PubMed abstracts and PubMed Central full-text articles, giving it deep biomedical vocabulary understanding.

Fine-tuned for Cardiology

We further fine-tune the model on our cardio-specific dataset for 5 epochs, adjusting the [CLS] classification head to output 3-class predictions.

HuggingFace Transformers

Implemented using dmis-lab/biobert-base-cased-v1.2 from the HuggingFace Model Hub via the transformers library.

Training

Training Pipeline

1. Load Dataset

Read CSV → split 80/10/10 for train, validation, test sets.

2. Tokenize

BioBERT tokenizer with max_length=256 and dynamic padding.

3. Fine-Tune

5 epochs, AdamW optimizer (lr=2e-5), batch size 16, linear warmup.

4. Evaluate

Accuracy, F1-macro, precision, recall on held-out test set.

5. Save Model

Best checkpoint saved via save_pretrained() for inference.

6. Serve via Flask

Model loaded at startup; predictions served over /api/predict.
